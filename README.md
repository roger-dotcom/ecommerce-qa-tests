# E-commerce QA Tests – Playwright

End-to-end QA automation tests for an e-commerce application using Playwright.
Includes both positive and negative test cases:
* Login (success & failure)
* Cart functionality (add/remove items)
* Checkout flow
* Price validation
Built with reusable functions and a structured test architecture.
Tech: Playwright, JavaScript, Git

## Tested features

- User login
- Add item to cart
- Remove item from cart
- Checkout flow
- Total price validation
- Invalid login error handling

## Technologies

- Playwright
- JavaScript
- Node.js

## How to run tests

```bash
npm install
npx playwright test
```

## What this project demonstrates
- End-to-end testing using Playwright
- UI automation for e-commerce flows
- Test coverage for login, cart, checkout
- Error handling validation
