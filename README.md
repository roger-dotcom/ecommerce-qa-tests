# E-commerce QA Tests – Playwright

End-to-end QA automation tests for an e-commerce demo website using Playwright.

## Tested features

- User login
- Add item to cart
- Remove item from cart
- Checkout flow
- Total price validation
- Invalid login error handling

## Technologies

- Playwright
- JavaScript
- Node.js

## How to run

```bash
npm install
npx playwright test
```
